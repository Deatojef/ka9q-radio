Enumerate the HID Devices. 

This function returns a linked list of all the HID devices attached to the system which match vendor\+\_\+id and product\+\_\+id. If {\ttfamily vendor\+\_\+id} is set to 0 then any vendor matches. If {\ttfamily product\+\_\+id} is set to 0 then any product matches. If {\ttfamily vendor\+\_\+id} and {\ttfamily product\+\_\+id} are both set to 0, then all HID devices will be returned.


\begin{DoxyParams}{Parameters}

\hline
{\em product\+\_\+id} & The Product ID (PID) of the types of device to open.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
This function returns a pointer to a linked list of type struct \doxylink{hidapi_8h_aa6da74d5686d198dd3e5440e60088fcc}{hid\+\_\+device}, containing information about the HID devices attached to the system, or NULL in the case of failure. Free this linked list by calling \doxylink{group__API_ga32d26a69bd5a374d7e856e7c5dd164dc}{hid\+\_\+free\+\_\+enumeration()}. 
\end{DoxyReturn}

Initialize the HIDAPI library. 

This function initializes the HIDAPI library.
